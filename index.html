Front-end development is the process of designing and building a website's graphical user interface (GUI) so that users can interact with it. Web design is the process of creating landing pages, fixing user interface bugs, and testing and maintaining websites.

Front-end development courses teach students how to create interactive websites by building the visual layout and user interface. Web development courses teach students how to develop websites for the internet or intranets.
